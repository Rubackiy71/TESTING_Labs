Файл с описанием тестов.
-----------
_A_ _B_ _C_ y
-----------
где _A, _B _C - числа из входного файла;
    y - '0'=true или '1'=false - ожидаемый ответ программы

##################################
ожидаемая работа программы:
    1.прочитан тест. Перейти к 2.
    2. вывод ответа на тест. перейти к 1.
